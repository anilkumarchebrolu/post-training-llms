# LLM Post-Training

This repository contains hands-on experiments and implementations of post-training techniques for Large Language Models (LLMs).

The goal of this project is to practically explore and understand how LLMs can be customized and aligned after pre-training using modern post-training methods.

---
## Focus
- Supervised Fine-Tuning (SFT)  
- Direct Preference Optimization (DPO)  
- Reinforcement Learning (PPO / GRPO)  

---
## Goal
Build hands-on understanding of how LLMs are customized, aligned, and improved after pre-training.

---
## Approach
- Small open-source models  
- Lightweight, reproducible experiments  
- Clean implementations  
- Continuous iteration
